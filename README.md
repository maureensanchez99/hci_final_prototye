# MediMate

MediMate is a Flutter app that helps users manage their medications by scheduling reminders and sending notifications.

## Prerequisites

Before running this app, make sure you have the following installed:

- [Flutter SDK](https://docs.flutter.dev/get-started/install) (version 3.0 or higher)
- Android Studio (for running on emulators)
- A code editor (VS Code or Android Studio recommended)

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd hci_final_prototye
   ```

2. Install dependencies:
   ```bash
   flutter pub get
   ```

## How to Run

1. Connect a physical device or start an emulator

2. Run the app:
   ```bash
   flutter run
   ```

3. For a specific platform:
   ```bash
   flutter run -d android    # For Android
   flutter run -d ios        # For iOS
   ```

## Features

- Add and manage medications
- Schedule medication reminders
- Receive notifications for medication times
- Track medication history
- Estimate refill dates
